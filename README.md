# h5协议自动化前端

基于 Vue3 + Vite + Element Plus 的协议管理系统前端。

## 功能特性

- ✅ 协议文件列表展示
- ✅ 协议文件的创建、编辑、删除
- ✅ 富文本编辑器支持（支持格式和样式编辑）
- ✅ 实时预览

## 技术栈

- Vue 3
- Vite
- Element Plus
- Axios

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

访问 http://localhost:3000

## 构建生产版本

```bash
npm run build
```

## 说明

### 关于编辑功能

协议编辑支持两种模式：

1. **代码编辑模式**（当前实现）：直接编辑 HTML 源码，可以完全控制格式和样式
2. **富文本编辑模式**（可扩展）：使用 TinyMCE 等富文本编辑器，可视化编辑内容

建议：
- 对于需要精确控制格式的场景，使用代码编辑模式
- 对于只需要修改文字内容的场景，可以扩展富文本编辑器功能
